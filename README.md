# Event Management Dashboard

This is my solution for event management dashboard problem

### Problem Statement:
Build an event management dashboard where organizers can create
and manage events, and users can browse and register for events.
The dashboard should include features like event creation, user
registration, event listings, and real-time updates on the number of
registrations.
### Key Requirements:
- Frontend: Design a responsive dashboard for both organizers
and users, with different views based on user roles.
- Backend: Implement event management, user authentication,
and real-time updates for event registrations.
- Database: Store event details, user information, and
registration data.
- Bonus: Include a notification system to alert users about
upcoming events or registration deadlines.
